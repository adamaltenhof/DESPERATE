# DEnoising SPEctRA in pyThon with wavElets (DESPERATE)

DESPERATE is a Python library for processing and denoising NMR spectra


## Requirements
- All testing has been done with Python 3.10 and newer
- NumPy
- SciPy
- matplotlib
- PyWavelets
- nmrglue (mainly optional)
